# Biopol Chain Reaction Mobile

Biopol Chain Reaction Mobile is the standalone touch-first build of the polymer survival game, packaged as an independent GitHub Pages site for iPad and phone testing.

## What it includes

- Standalone mobile-first layout for GitHub Pages
- Touch-safe arena with swipe input and turn pads
- Configurable human and CPU chain slots, including extra CPU chains
- Compact play mode that hides setup chrome during active rounds
- Multiple arena layouts and score targets

## How to run it

Open [index.html](./index.html) directly in a browser.

If you prefer serving it locally:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Touch Controls

- Swipe on the arena to steer chain 1
- Tap `Turn left` or `Turn right` for one-handed play
- Use the setup drawer before a round to enable manual or CPU chains
